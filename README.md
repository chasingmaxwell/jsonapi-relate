## Installation
```
npm install jsonapi-relate --save
```

## Development
### Testing
Using Mocha/Chai for testing.
```
npm t
```

### Code Coverage
Generate a report using Istanbul to make sure your tests are touching everything!
```
npm t && npm run view-coverage
```
